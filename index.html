Smart Automation Features (SCADA-IoT)

1. Real-Time Dashboard (SCADA-IoT)

  • • Pelacakan real-time pH, TDS, Turbiditas, COD, dan stok reservoar secara otonom.
  • • Visualisasi Level OWD (Over Water Depth) dengan batas peringatan kritis 330-350 mm.
  • • Status Operasional: Green (Running), Red (Fault), Grey (Standby) dengan sistem Running Hours.

2. Auto-Dosing Adjustment (SCADA-IoT Driven)

  • • Sinkronisasi Aliran: SCADA-IoT membaca sinyal 4-20mA dari flow meter intake secara presisi.
  • • Kecepatan dosing otomatis skala naik/turun sesuai volume intake melalui kontrol cerdas.
  • • Koreksi Turbiditas: Peningkatan PPM otomatis seketika saat sensor mendeteksi kekeruhan tinggi.

3. Auto-Backwash (SCADA-IoT Sequential Control)

  • • Deteksi Head Loss otomatis pada 350 mm melalui algoritma SCADA-IoT.
  • • Sekuens Pembersihan: Proses Drawdown, Air Scour, Water Wash, dan Settling tanpa intervensi.
  • • Filter-to-Waste: Pengalihan otomatis air awal pasca-backwash untuk menjamin kejernihan.

4. Automated Batching (Stock Management)

  • • Interlock Stok Cerdas: Sistem berhenti otomatis melalui SCADA-IoT jika stok larutan < 10%.
  • • Batching Otonom: Pemicu otomatis siklus pengisian air, penimbangan bubuk, dan Aging Phase.

5. Smart Fail-Over & Safety Protection

  • • Auto-Swap: Pengaturan pergantian pompa setiap 8-12 jam melalui sistem SCADA-IoT.
  • • Automatic Fail-Over: Aktivasi instan pompa standby jika sistem mendeteksi kegagalan unit aktif.
  • • Proteksi Mekanis: Sistem Anti-Water Hammer dan Dry-Run melalui kontrol delay cerdas.
